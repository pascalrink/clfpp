# Confidence-Limits-for-Prediction-Performance

This repository contains the code and supporting materials to my PhD thesis "Confidence Limits for Prediction Performance", available at [https://doi.org/10.26092/elib/3822](https://doi.org/10.26092/elib/3822). The project was originally hosted on the university's GitLab instance at [https://gitlab.informatik.uni-bremen.de/s_opbgf3/clfpp](https://gitlab.informatik.uni-bremen.de/s_opbgf3/clfpp) and has been migrated here to preserve the work and make the implementation publicly accessible.

The repository is provided primarily for documentation and reproducibility purposes. It reflects the state of the project at the end of the research phase and is not under active development.

## Status

**Archived** – This repository is maintained as a historical snapshot of the research codebase. No further updates or maintenance are planned.

## Origin

The project was originally developed as part of my doctoral research at the university and was transferred from the institutional GitLab after completion of the dissertation to ensure long-term availability.

## Usage

The code is made available for reference and research purposes. While the repository should allow others to understand the implementation details behind the dissertation, compatibility with current software environments is not guaranteed.
